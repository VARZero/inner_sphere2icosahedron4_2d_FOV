# inner_sphere2icosahedron4_2d_FOV
2D 이미지들을 방위각/위도에 맞게 구 공간에 투영하고, 투영된 구 이미지가 HEALpix 형태로 픽셀로 변환되고, 이 변환된 픽셀을 정이십각형에 투영하여 버퍼에 저장하고, 정이십각형 삼각형 중에 시선에 들어오는 삼각형을 찾아 영상신호로 바꾸어, 평면 디스플레이에 출력하는 프로젝트입니다.<br>
Zynq-7020에서 출력 및 투영 가속(2D 이미지들이 투영된 정이십각형 버퍼 생성 및 삼각형 추출과 영상신호 생성) 설계된 HDL을 PL에 구동하고, 이를 제어하기위한 리눅스 드라이버와 API, 예제 프로그램을 설계하는 것이 이 프로젝트의 구현 목표입니다.<br>
